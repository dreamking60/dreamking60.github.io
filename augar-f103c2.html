Augar安装和配置

Posted on Sun, Jan 2, 2022 tool Taint Analysis

augur是一个针对javascript的动态污点分析工具。我的创新实践项目就是研究动态污点分析，我们选定的方向是web应用程序的动态污点分析，我们需要接触一些工具，augur是少数的正在维护的动态污点分析工具，且它的安装使用相对来说比较简单，因此我们选用了这个工具来进行初步的研究。 仓库地址：https://github.com/nuprl/augur 注意：按照我的方式安装是使用了他们提供的docker，那个docker里面已经搭好了相关nodejs的环境，如果你想用自己的环境，可以根据Readme.md下面的教程进行搭建。

augur的安装推荐使用ubuntu20.04虚拟机，需要提前安装的docker，安装docker使用菜鸟教程的一键安装脚本即可。 地址：https://www.runoob.com/docker/ubuntu-docker-install.html 注意：curl需要先提前安装，按照提示信息来即可。 我使用的是：

curl -fsSL <https://get.docker.com> | bash -s docker --mirror Aliyun

当docker安装完成后，按照仓库地址的提示安装即可。

git clone --recurse-submodules <https://github.com/nuprl/augur>
cd augur/ts

注意：为了能够运行node，我们需要一个较高版本的node，因此我们不能直接sudo安装npm，应该使用下面的命令换源进行安装。

curl -sL <https://deb.nodesource.com/setup_14.x> | sudo -E bash -
sudo apt update
sudo apt install nodejs

注意运行下面的命令需要用sudo，不用sudo可能会有permission的错误信息，出现了前面加sudo即可。

sudo ./docker-nodeprof/docker-pull.sh  # Pull NodeProf Docker image
npm install                       # Install Augur deps
npm run build                     # Build Augur

跑测试可以按照下面的代码，根据相关信息更换案例进行测试。 注意：可能需要使用sudo。

./node_modules/.bin/jest -t basic-assignment-tainted

解压文件

tar -zxvf *.tar.gz
