Rust is an ahead-of-time compiled language. Everything is a trade-off in language design.

Compile and Execute

fn main() {
	println!("Hello, world!");
}
#complie
rustc main.rs
#exectute
./main

Build project by cargo

#init project
cargo new [project name]
#compile project
cargo build
#run the project
cargo run
#check the project's executable file
cargo check
# build for release
cargo build --release
